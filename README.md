# 2D шутер про зомби

2D шутер про зомби с видом сверху, где игроку предстоит выжить
в условиях постоянного давления со стороны нежити.
Волны становятся сложнее, врагов больше, а ресурсов всё меньше —
приходится выбирать, как сражаться и когда отступать.

## ⚙️ Реализовано в проекте
- Динамичные волны зомби
- Система урона и смерти
- Стрельба и боезапас
- Интерфейс игрока
- Визуальные эффекты
